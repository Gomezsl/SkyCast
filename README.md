# 🌤️ SkyCast - Dashboard de Clima Dinámico

¡Bienvenido a SkyCast! Una aplicación web moderna que muestra el clima en tiempo real utilizando la API de OpenWeatherMap.

## 🚀 Características
* **Búsqueda global:** Consulta el clima de cualquier ciudad del mundo.
* **Interfaz Adaptativa:** El fondo y los iconos cambian según el estado del tiempo (Sol, Nubes, Lluvia).
* **Diseño Glassmorphism:** Estilo moderno con efectos de desenfoque y transparencia.
* **Geolocalización:** (Próximamente) Detección automática de la ubicación del usuario.

## 🛠️ Tecnologías Usadas
* **HTML5** & **CSS3** (Flexbox/Glassmorphism)
* **JavaScript** (Fetch API & Async/Await)
* **OpenWeather API**

## 📦 Instalación
1. Clona el repositorio: `https://gomezsl.github.io/SkyCast/`
2. Abre `index.html` en tu navegador.
