Sobre el Proyecto

SkyCast es una aplicación web intuitiva diseñada para la consulta dinámica del clima en tiempo real. Este proyecto combina una interfaz limpia con funcionalidades avanzadas como geolocalización y pronósticos extendidos para ayudar a los usuarios a planificar su día.

Este proyecto te permite conocer el estado del tiempo desde cualquier lugar con datos precisos y visuales dinámicos.

Meta: Consolidar una herramienta meteorológica robusta y sencilla de usar.
